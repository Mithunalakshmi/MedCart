# 🏥 MedCart – Medical Equipment E-Commerce Platform

MedCart is a full-stack medical equipment e-commerce platform designed to simplify the process of purchasing medical devices online. The application provides a secure, centralized, and user-friendly platform where individuals, clinics, and healthcare providers can browse, compare, and purchase genuine medical equipment with ease.

---

## 🚀 Features

- 🔐 Secure User Authentication (JWT)
- 👤 User Registration & Login
- 🩺 Browse Medical Equipment
- 🔍 Product Search & Filtering
- 🛒 Shopping Cart Management
- ❤️ Wishlist Management
- 📦 Order Placement & Management
- 👤 User Profile Management
- 🔒 Protected Routes
- 📱 Responsive User Interface
- ⚡ RESTful APIs

## 🎯 Problem Statement

Purchasing medical equipment is often difficult due to fragmented vendors, inconsistent pricing, limited product availability, and time-consuming offline processes. MedCart addresses these challenges by providing a secure, centralized, and easy-to-use platform for buying medical equipment online.

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/MedCart.git
```

### Navigate to the Project

```bash
cd MedCart
```

### Install Dependencies

#### Frontend

```bash
cd frontend
npm install
```

#### Backend

```bash
cd ../backend
npm install
```

### Configure Environment Variables

Create a `.env` file inside the `backend` folder and add:

```env
PORT=5000
DB_HOST=localhost
DB_USER=your_mysql_username
DB_PASSWORD=your_mysql_password
DB_NAME=medcart
JWT_SECRET=your_secret_key
```

### Run the Backend

```bash
npm start
```

### Run the Frontend

```bash
cd ../frontend
npm start
```
